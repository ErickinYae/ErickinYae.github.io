IP地址

  • 私有地址：私有地址分为三类 -> A类地址、B类地址和C类地址；

    • A类：10.0.0.0 到 10.255.255.255
    • B类：172.16.0.0 到 172.31.255.255
    • C类：192.168.0.0 到 192.168.255.255

    三类地址的区别主要在于分配的私有地址数量，A类最大、C类最少。

  • 公有地址：公网IP地址的分配和管理是由全球的互联网资源分配机构进行的，对格式没有明确的规定；公网IP地址必须在全球范围内是唯一的，以确保在互联网上的设备可以正确地标识和通信。

协议

  • DNS协议；
  • ICMP协议；
  • TCP/IP协议；
  • UDP协议；
  • HTTP协议；
  • HTTPS协议。

端口

默认可使用端口1-65535个端口

协议、应用的默认通信端口

  • DNS协议：53；
  • HTTP协议：80；
  • HTTPS协议：443；
  • Apache、Nginx、IIS：80；
  • Tomcat：8080；
  • MYSQL：3306；
  • SQL server：1433；
  • Oracle：1521；

Nginx

配置文件：.\conf\nginx.conf；

访问127.0.0.1这个地址，是怎么将页面展示出来的？

访问的页面实际上对应着一个静态文件叫做index.html，目录在/html/index.html.

浏览器会将请求返回得到的html进行解析，并对其中的css、js进行处理和运行。

nginx.conf

页面重定向

    location / {
        #网站根目录
        root   "E:/WWW/www/";
        #网站默认首页文件名称
        index  index.html 1.txt;
    }  
    
    location ^~/bbs$ {
        #网站根目录
        alias   "E:/WWW/bbs/";
        #网站默认首页文件名称
        index  index.html 1.txt;
    }
